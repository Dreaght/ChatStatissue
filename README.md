# ChatStatissue
Simple web-socket chat application.

> [!NOTE]
> To open chat press `alt+X`.
> To close - `ESC` or `Enter` to send a message.

# Build and run
```./gradlew run```

# Compile an executable file
```./gradlew jpackage```
